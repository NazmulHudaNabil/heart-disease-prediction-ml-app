# 🫀 Heart Disease Prediction ML Web App

## 📌 Overview

This project is a Machine Learning-based Heart Disease Prediction Web Application built using **Scikit-Learn** and **Streamlit**.

The model was trained on a heart disease dataset and achieved an **F1-score of 91%**.

The application allows users to input patient medical data and receive a real-time prediction of heart disease risk.

---

## 🚀 Features

* User-friendly Streamlit web interface
* Real-time heart disease risk prediction
* Machine Learning model (trained & optimized)
* 91% F1-score
* Clean and deployable project structure

---

## 🧠 Machine Learning Workflow

1. Data Cleaning
2. One-Hot Encoding
3. Feature Scaling
4. Model Training
5. Model Evaluation
6. Model Saving (Joblib)
7. Streamlit Web Deployment

## ▶️ How To Run Locally

1. Clone the repository:

```
git clone https://github.com/NazmulHudaNabil/heart-disease-prediction-ml-app.git
```

2. Navigate into the folder:

```
cd heart-disease-prediction-ml-app
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run the app:

```
streamlit run app.py
```
